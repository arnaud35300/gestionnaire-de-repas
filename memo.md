# install tailwind

see : https://medium.com/better-programming/tailwindcss-and-symfonys-webpack-encore-7bfc8c18665b

and add :
```
yarn add --dev \
  postcss-loader \
  postcss-import \
  postcss-cssnext \
  cssnano \
  sugarss \
  autoprefixer
```

see : https://github.com/postcss/postcss-loader/issues/311

end with :
`yarn run dev` or `yarn run build`
